Nitte Open Source Conference 2016

To spread open source awareness in Engineering Colleges in and around
Mangalore, India and inspire students to contribute to the open source movement.
Students will have an oppurtunity to learn from and interact with members
of different open source communities. It is also a platform for the students to
share new innovative ideas/concepts with community members.

"Open Think Innovate"

Topics

Programming Language

Talks on new programming languages or exciting features in exisitng programming languages

Storage

Talks on data storage systems, technolgies or concepts.

Virtualization

Talks on virtualization techologies like containers, kvm etc.

Operating System

Talks on operating systems.

Cloud

Talks on cloud technologies.

Graphics

Talks on graphics such as OpenGL.
